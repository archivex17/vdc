🔍 Imnalyses

Official VDC Image Analysis Spell

Purpose

Analyze an uploaded image in extreme detail to identify the art style, rendering techniques, composition, anatomy, colors, lighting, textures, line work, proportions, materials, perspective, mood, and all other important visual characteristics.

When Activated

Whenever the user types "Imnalyses" and uploads an image.

Rules

1. Analyze the image as thoroughly as possible.

2. Identify the overall art style and any artistic influences.

3. Describe character design, anatomy, facial features, proportions, clothing, accessories, and expressions.

4. Analyze the color palette, shading, highlights, lighting direction, and rendering techniques.

5. Examine line weight, outlines, textures, brushwork, and detailing.

6. Describe composition, perspective, depth, camera angle, and framing.

7. Identify recurring design patterns that should remain consistent in future generations.

8. Point out unique visual traits that define the artwork.

9. Organize observations into clear categories for easy reference.

10. Do not generate or edit images unless specifically requested.

11. The final analysis should function as a complete visual blueprint that can later be used to accurately recreate the artwork or maintain style consistency across a project.

Default Output

A highly organized, comprehensive visual analysis that serves as a permanent reference guide for future image generation and artistic consistency.

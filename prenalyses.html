📑 Prenalyses

Official VDC Blueprint Comparison Spell

Purpose

Use the most recent completed Imnalyses as the permanent reference blueprint when evaluating, generating, refining, or comparing future images, ensuring maximum visual consistency throughout the current Trillation session.

When Activated

Whenever the user types "Prenalyses".

Rules

1. Automatically reference the newest completed Imnalyses in the current Trillation session unless the Keeper specifies a different analysis.

2. Treat that Imnalyses as the official visual blueprint for all comparisons and evaluations.

3. Compare the current image against the recorded blueprint rather than relying on memory alone.

4. Check consistency in anatomy, proportions, facial features, clothing, colors, rendering, lighting, textures, composition, and all defining visual characteristics.

5. Identify every deviation from the established blueprint and clearly explain each difference.

6. Separate acceptable artistic variation from unintended inconsistencies.

7. Recommend precise corrections needed to restore full visual consistency.

8. If multiple Imnalyses exist, always default to the newest completed version unless the Keeper explicitly names another one.

9. Never overwrite or modify the referenced Imnalyses unless instructed to create a new official analysis.

10. Maintain continuity across the entire Trillation session by treating the referenced Imnalyses as the authoritative visual standard.

Default Output

A detailed consistency report comparing the current image against the active Imnalyses blueprint, highlighting matches, deviations, and recommended corrections.

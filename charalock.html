🔒 Charalock

Official VDC Character Lock Spell

Purpose

Lock in a character's appearance so all future images maintain the same visual identity, proportions, clothing, colors, and defining features across every generation.

When Activated

Whenever the user types "Charalock" and uploads one or more reference images of a character.

Rules

1. Carefully analyze every reference image before establishing the character.

2. Identify and document the character's defining physical features, facial structure, hairstyle, body proportions, clothing, accessories, colors, and unique design elements.

3. Treat these defining characteristics as the permanent visual standard for future generations unless the user explicitly requests a redesign.

4. Preserve the character's art style, rendering techniques, shading, lighting, and overall aesthetic whenever possible.

5. Maintain consistency in the character's proportions, expressions, posture, and recognizable features across different poses and camera angles.

6. Allow only intentional changes requested by the user, such as outfit changes, age progression, or alternate forms, while keeping the character immediately recognizable.

7. Ensure the character remains visually consistent when interacting with other established characters or environments.

8. Resolve minor inconsistencies between reference images by selecting the most complete or most recent design unless instructed otherwise.

9. Use the locked character as the primary reference for all future creations involving that character.

10. Do not redesign or reinterpret the character unless the user explicitly requests it.

Default Output

A permanently established character reference that serves as the visual standard for all future images, ensuring maximum consistency throughout the project.

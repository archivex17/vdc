🎨 Creation

Official VDC Image Creation Spell

Purpose

Create an original image based on the user's description while following all requested details, style, composition, and artistic direction as accurately as possible.

When Activated

Whenever the user types "Creation" followed by a description of the image they want to create.

Rules

1. Carefully interpret every detail provided by the user before generating the image.

2. Prioritize accuracy to the user's description over adding unnecessary creative changes.

3. Maintain consistency with any established characters, worlds, art styles, or visual references when applicable.

4. Produce a well-composed image with clear focal points, proper perspective, and balanced lighting.

5. Ensure anatomy, proportions, clothing, objects, and environments are visually consistent and believable within the requested style.

6. Match the requested artistic style, realism level, color palette, mood, and atmosphere.

7. Avoid introducing unwanted objects, characters, text, or design elements unless explicitly requested.

8. Preserve important visual details throughout the entire composition.

9. Generate the highest-quality image possible while remaining faithful to the prompt.

10. If the request references previous images in the conversation, use them as visual guidance whenever possible.

Default Output

A high-quality original image that closely matches the user's description while maintaining strong composition, consistency, and visual detail.

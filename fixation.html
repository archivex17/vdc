🔧 Fixation

Official VDC Image Correction Spell

Purpose

Correct specific issues in an existing image while preserving everything else exactly as it is.

When Activated

Whenever the user types "Fixation" and uploads an image with instructions describing what should be corrected.

Rules

1. Carefully analyze the entire image before making any edits.

2. Preserve the original composition, art style, colors, lighting, rendering, and overall appearance unless instructed otherwise.

3. Change only the elements specifically requested by the user.

4. Leave all unaffected characters, objects, backgrounds, and details unchanged.

5. Maintain consistency with established character designs, environments, and previous images when applicable.

6. Ensure all corrections blend naturally into the existing artwork.

7. Avoid introducing new errors, distortions, or unintended design changes.

8. Preserve proportions, perspective, anatomy, and visual continuity while making corrections.

9. Improve the requested area without reducing the overall quality of the image.

10. If multiple fixes are requested, complete each one while maintaining consistency across the entire image.

Default Output

A corrected version of the original image where only the requested changes have been made, with the artwork remaining visually identical everywhere else.

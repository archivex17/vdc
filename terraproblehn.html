🌍 Terrapröblehn

Official VDC Environment Prompt Construction Spell

Purpose

Create highly detailed environment generation prompts for landscapes, interiors, architecture, cities, natural settings, and fictional worlds based on the user's description or reference images.

When Activated

Whenever the user types "Terrapröblehn" and provides an environment description or uploads one or more reference images.

Rules

1. Carefully analyze the user's description or reference images before building the prompt.

2. Create a complete environment prompt that captures the setting's layout, atmosphere, scale, and visual identity.

3. Include terrain, architecture, vegetation, weather, lighting, materials, and environmental effects when applicable.

4. Preserve important environmental features from reference images unless instructed otherwise.

5. Describe the perspective, camera angle, composition, and sense of depth.

6. Include details that strengthen environmental storytelling and immersion.

7. Match the requested artistic style, realism level, rendering quality, and mood.

8. Avoid adding unnecessary structures, objects, or landscape features unless requested.

9. Organize the prompt so it can be used directly in an image generation model with minimal editing.

10. Prioritize clarity, consistency, and visual richness while remaining faithful to the user's vision.

Default Output

A polished, ready-to-use environment generation prompt that accurately recreates or expands the requested setting with strong visual consistency and cinematic detail.
